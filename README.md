# Make_up_Study
小白学习化妆资料

# 课程大纲

- 理论课：方法，产品，基础，审美。
  化妆公式；选购合适的化妆品；护肤和美妆常识；如何正确分析自己和美学原理在化妆中的应用。
- 局部妆：底妆，眉妆，眼妆，唇妆，综合。
- 场合妆：七个主题妆容。
  学生妆，约会妆，出游妆，芭比妆，职场的亲和妆，晚宴妆。
- 附加：发型课，番外篇。

# 学习思想认识

- 不当装备党
  化妆 = 50%审美 + 30%技巧 + 20%产品
- 化妆不是万能（针对面部的平面修饰）
- 学习过程：不出错---> 有变化 ----> 有个性
- 一物多用，不要死板：即使是口红也可以当作眼影，腮红，遮瑕
